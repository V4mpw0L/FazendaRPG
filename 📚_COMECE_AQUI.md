## ⚡ Sistema de Energia (v0.0.5)

### O Básico
```
Cada nível = +5 Energia Máxima
Cada skill level = +5 Energia Máxima
```

### Exemplos
```
Level 1, Skills 1:     100/100
Level 5, Skills 1:     125/125
Level 10, Skills 5:    305/305
```
